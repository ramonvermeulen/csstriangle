About CSS Triangle Generator

CSS Triangle Generator started off as a personal freetime summer project to get more familiar with the Angular 6 framework. After I finished the project I thought it was nice to put it online so other people globally could make use of it.

The main goal of CSS Triangle Generator is to provide an easy to use interface which generates CSS code for a triangle. To use the generated triangle on your own website or application all you have to do is put a <div class="triangle"><div> in your HTML file and copy / paste the generated CSS code into your stylesheet.

If you walk into any bugs or if you have any suggestions, please submit an issue through github or feel free to mail me at the e-mail address below.
